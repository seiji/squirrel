# Squirrel

SQLite client command line

## Installation

Checkout this project:

    hub clone seiji/squirrel

And then execute:

    $ bundle

## Usage

Execute this script

    squirrel /path/to/your.sqlite

Or

    bundle exec bin/squirrel /path/to/your.sqlite
